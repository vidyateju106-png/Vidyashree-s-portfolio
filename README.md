# Web Developer Portfolio

A personal portfolio website showcasing skills, projects, and education. This static website is designed to be responsive, visually appealing, and easy to navigate, serving as a digital resume for a web developer.

## Features

-   **Responsive Design**: Fully responsive layout that adapts to mobile, tablet, and desktop screens.
-   **Particle Background**: Interactive particle background animation using HTML5 Canvas.
-   **Typing Effect**: Dynamic typing text effect in the hero section.
-   **Smooth Scrolling**: Smooth navigation between sections.
-   **Scroll Animations**: Elements fade in and slide up as you scroll down the page.
-   **Glassmorphism UI**: Modern glassmorphism design style for cards and containers.
-   **Interactive Elements**: Hover effects on buttons, cards, and navigation links.

## Technologies Used

-   **HTML5**: Semantic markup for structure.
-   **CSS3**:
    -   Flexbox and Grid for layout.
    -   CSS Variables for theming.
    -   Keyframe animations and transitions.
    -   Media queries for responsiveness.
-   **JavaScript (Vanilla)**:
    -   DOM manipulation.
    -   `IntersectionObserver` API for scroll animations.
    -   Canvas API for background particles.
-   **Font Awesome**: For icons (social media, skills, contact).
-   **Google Fonts**: 'Inter' and 'Outfit' fonts for typography.

## Setup / Installation

Since this is a static website (HTML, CSS, JavaScript), you don't need to install any complex software or servers to run it.

### Option 1: The Easiest Way
1.  Navigate to the folder where you saved these files.
2.  Locate the `index.html` file.
3.  **Double-click** `index.html`. It will open in your default web browser (like Chrome, Edge, or Firefox).

### Option 2: Using VS Code (Recommended for editing)
1.  If you have VS Code installed, open this folder in VS Code.
2.  Install the "Live Server" extension if you want to see changes instantly.
3.  Right-click `index.html` and select "Open with Live Server".

## Project Structure

-   **index.html**: The main content of your website.
-   **style.css**: Controls the look and feel (colors, fonts, layout).
-   **script.js**: Handles interactivity (mobile menu, animations, particles).
